# Assets для SEO

Эта папка содержит изображения для Open Graph и Twitter Cards.

## Требуемые изображения

### Open Graph изображения (минимум 1200x630px):
- `og-image-ru.jpg` - для русской версии
- `og-image-en.jpg` - для английской версии

### Twitter изображения (рекомендуется 1200x675px):
- `twitter-image-ru.jpg` - для русской версии  
- `twitter-image-en.jpg` - для английской версии

## Рекомендации по созданию изображений

1. **Размер**: Минимум 1200x630px для OG, 1200x675px для Twitter
2. **Формат**: JPG или PNG
3. **Содержание**: 
   - Логотип или название проекта
   - Краткое описание (опционально)
   - Привлекательный дизайн, соответствующий бренду
4. **Текст**: Избегайте мелкого текста, так как изображения могут масштабироваться

## Где используются

- Open Graph изображения используются при шаринге в Facebook, LinkedIn и других соцсетях
- Twitter изображения используются при шаринге в Twitter/X
